# Rock-Paper-Scissor-GAME
This is a console-based Rock-Paper-Scissors game developed in C++. 
The program allows a user to play against the computer in a simple and interactive way.

🎮 How It Works
The user selects one of the options: Rock, Paper, or Scissors.
The computer randomly generates its choice.
The program compares both choices and determines the winner based on standard game rules:
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
The result is displayed instantly in the console.

⚙️ Features
Simple and user-friendly interface
Randomized computer moves
Instant result display
Lightweight and runs in any C++ console environment
